# FOODHUB

Foodhub:  is a Food Delivery Application that has Features like Tracing your order, search filter, nearest Restaurant ...etc

Technology Used:
● Retrofit
● Firebase
● Preferences
● Fragment
● Navigation
● RecyclerView
● Material Design
● MVVM
● Map
